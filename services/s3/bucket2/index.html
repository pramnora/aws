This is my first Amazon s3 web site.

Web site creation process

Creating it was really pretty straight forwards...

  • Go to the Amazon console: aws.amazon.com
  • Select service: S3
  • Give the bucket a name
  • Choose access control...in the case of a public web site, turn off option private.
  • Add a couple of tags: name/value used to ID the bucket with
  • Next, select option: [Create bucket]
  • Once the bucket has been created...select it from the list...and, upload web page file/(s): [index.html]
  • Choose Presigned URL...and, set the URL to expire in 12 hours time.
  • Finally, click option: [Open]; and, the web browser will display your brand 'new' web page.
